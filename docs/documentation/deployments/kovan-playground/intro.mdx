# Kovan playground

Gearbox supports deployment on Kovan network as experimetal way to check composability. Kovan playground is ready-made testnet
which has tokens and contract mocks which synced with Mainnet. So, as a developer, you can expect similar courses and ways to interact

:::info
Please, check Kovan playground github: [https://github.com/Gearbox-protocol/kovan-playground](https://github.com/Gearbox-protocol/kovan-playground)
:::

## Mainnet sync
Prices and other mainnet data is sync by robot. All deployed mocks contracts and price updaters could be updated by
address with syncer role (check [`Sycner.sol`](https://github.com/Gearbox-protocol/kovan-playground/blob/main/contracts/Syncer.sol), 
deployed at [0xC6493381d29e813D56063A1AffBbbC534fdCd70b](https://kovan.etherscan.io/address/0xC6493381d29e813D56063A1AffBbbC534fdCd70b#code))
